# Markdown 语法

## 标题

```js
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题

## 字体

```js
**加粗字体**
*斜体字体*
***加粗并且倾斜字体***
~~删除线字体~~
```
**加粗字体**  
*斜体字体*  
***加粗并且倾斜字体***  
~~删除线字体~~

## 列表

#### 无序列表

`- + * 都可以生成无序列表 注意符号和内容之间有空格`

+ 第一条
- 第二条
* 第三条

#### 有序列表

`有序列表 注意序号和内容之间有空格`

1. 滴滴滴
2. 哒哒哒
3. 咯咯咯

- 工作内容
   1. 看邮件
   2. 写代码
   3. 开会
- 加油站
   - 看文章
   * 看书
   + 交流
